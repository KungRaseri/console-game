# New Game+ System

**Status**: See [IMPLEMENTATION_STATUS.md](../IMPLEMENTATION_STATUS.md)

## Overview

The New Game+ System allows players to replay the game with bonuses earned from completing the main quest while maintaining increased challenge.

## Core Components

### Starting Bonuses
- **Enhanced Attributes**: Improved starting stats for increased power
- **Resource Bonuses**: Increased starting HP, MP, and gold
- **Retained Progress**: Achievements and Hall of Fame records preserved
- **Early Advantages**: Access to features earlier than first playthrough

### Increased Challenge
- **Enemy Scaling**: Enemies scale beyond normal difficulty
- **Quest Requirements**: Objectives require greater effort
- **New Difficulty Options**: Unlock higher difficulty modes
- **Maintained Balance**: Challenge remains engaging despite bonuses

### Rewards and Incentives
- **Accelerated Progression**: Earlier access to high-tier items
- **Bonus Experience**: Faster leveling for quicker builds
- **Unique Recognition**: Special titles and achievements
- **Extended Content**: Encourages experimentation with builds
- **Endgame Replayability**: Provides post-victory content

## Key Features

- **Replay Value**: Encourages multiple playthroughs
- **Build Experimentation**: Try different approaches with bonuses
- **Completionist Appeal**: Unlock everything across playthroughs
- **Power Fantasy**: Play with advantages while facing worthy challenge
